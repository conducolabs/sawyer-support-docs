    Replace the stub in `src/commands/translate.ts` with full implementation. Mirror the generate command structure exactly.

    **Command definition:**
    ```
    translateCommand = new Command('translate')
      .description('Translate German articles to configured languages via DeepL')
      .option('--features <slugs>', 'Comma-separated feature slugs to translate')
      .option('--languages <langs>', 'Override config languages (comma-separated, e.g., en,nl)')
      .option('--dry-run', 'Show what would be translated and estimated character count')
      .option('--force', 'Overwrite translations even if manually edited')
    ```

    **Action implementation (inside .action(async (options) => { ... })):**

    1. `const config = loadConfig()` — get deepl_api_key and languages
    2. `const logger = createLogger(false, false)`
    3. `const cwd = process.cwd()`
    4. Read feature map: `readFeatureMap(cwd)` — exit with error if missing
    5. Filter features by `--features` if provided (same pattern as generate: split on comma, trim, filter by slug, warn if none match)
    6. Resolve target languages:
       - Start with `config.languages.filter(lang => lang !== 'de')` (always exclude German)
       - If `--languages` provided: parse comma-separated, validate each is in SUPPORTED_LANGS, intersect with filtered list
    7. Build work list: `features x targetLangs` — array of `{ feature, lang }` jobs
    8. **--dry-run path:**
       - For each feature: `readGermanArticle(cwd, feature.featureArea, feature.slug)` — skip if null
       - Parse frontmatter to get body, sum `body.length` across all jobs
       - Print each job: `"Would translate: {slug} -> {lang}"`
       - Print total: `"Estimated DeepL characters: {total}"`
       - `logger.success('Dry run complete — no translations made.')` then return
    9. **Translation loop:**
       - `const client = createDeepLClient(config.deepl_api_key)` (create ONCE before loop)
       - Track counters: `let translated = 0, skipped = 0, failed = 0`
       - For each feature (outer loop — sequential):
         - `const germanContent = readGermanArticle(cwd, feature.featureArea, feature.slug)`
         - If null: `logger.warn('No German source for {slug} — skipping')`, continue
         - `const parsed = parseFrontmatter(germanContent)`
         - `const sourceHash = computeHash(germanContent)`
         - For each targetLang (inner loop — sequential):
           - `const jobIndex = ++currentJob` (for progress display)
           - `const spinner = logger.spinner('Translating: {slug} ({lang}) ({jobIndex}/{totalJobs})...')`
           - Compute target absolute path: `resolve(cwd, buildArticlePath(lang, feature.featureArea, feature.slug))`
           - `const gating = checkGating(targetAbsPath, sourceHash, options.force ?? false)`
           - If `gating.action === 'skip'`: `spinner.succeed('Skipped: {slug} ({lang}) — hash unchanged')`, `skipped++`, continue
           - If `gating.action === 'warn_manual_edit'`: `spinner.warn('Skipped: {slug} ({lang}) — manually edited, use --force to overwrite')`, `skipped++`, continue
           - Try: `const translatedBody = await translateArticle(client, parsed.body, lang)`
           - Build full content: `buildTranslatedFrontmatter(parsed.title, lang, sourceHash) + translatedBody`
           - Write: `writeTranslatedArticle(cwd, lang, feature.featureArea, feature.slug, fullContent)`
           - `spinner.succeed('Translated: {relPath}')`, `translated++`
           - Catch: `spinner.fail(formatDeepLError(err, targetPath))`, `failed++`, continue
    10. Summary: `logger.success('Translation complete: {translated} translated, {skipped} skipped (hash unchanged), {failed} failed.')`
    11. `if (failed > 0) process.exit(1)`

    Wrap entire action body in outer try/catch for config errors (same as generate command).

    **Imports:**
    - `{ Command } from 'commander'`
    - `{ loadConfig } from '../config/index.js'`
    - `{ readFeatureMap } from '../scanner/index.js'`

    - `{ createDeepLClient, translateArticle, formatDeepLError, computeHash, checkGating, parseFrontmatter, buildTranslatedFrontmatter, readGermanArticle, writeTranslatedArticle } from '../translator/index.js'`
    - `{ buildArticlePath } from '../paths/index.js'`
    - `{ resolve } from 'node:path'`

    All imports use `.js` extensions (ESM project).
