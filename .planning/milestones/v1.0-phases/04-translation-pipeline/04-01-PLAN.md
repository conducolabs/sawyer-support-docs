    ## hash.ts
    - computeHash("hello") returns consistent SHA-256 hex string (64 chars)
    - computeHash("hello") === computeHash("hello") (deterministic)
    - computeHash("hello") !== computeHash("world")
    - checkGating(nonExistentPath, anyHash, false) returns { action: 'translate' }
    - checkGating(existingPath, sameHash, false) returns { action: 'skip', reason: 'hash_unchanged' }
    - checkGating(existingPath, differentHash, false) returns { action: 'warn_manual_edit' }
    - checkGating(existingPath, anyHash, true) returns { action: 'translate' } (force mode)
    - checkGating(existingPathNoSourceHash, anyHash, false) returns { action: 'translate' }

    ## frontmatter.ts
    - parseFrontmatter() extracts title, language, body from German source article
    - parseFrontmatter() extracts title, language, sourceHash, body from translated article
    - parseFrontmatter() throws on content without frontmatter delimiters
    - buildTranslatedFrontmatter("Login", "en", "abc123") produces ---\ntitle: "Login"\nlanguage: en\nsource_hash: abc123\n---\n\n
    - buildTranslatedFrontmatter handles title with quotes (escapes them)
    - Round-trip: buildTranslatedFrontmatter values can be read back by parseFrontmatter

    ## client.ts
    - translateArticle() calls client.translateText with sourceLang 'de', correct targetLang mapping, formality 'prefer_less', preserveFormatting true
    - Language mapping: en -> EN-US, nl -> NL, tr -> TR, uk -> UK
    - translateArticle() throws for unmapped language code
    - formatDeepLError maps AuthorizationError -> message containing "API key" and file path
    - formatDeepLError maps QuotaExceededError -> message containing "quota" and "deepl.com/account"
    - formatDeepLError maps unknown error -> message with error.message and file path

    ## writer.ts
    - readGermanArticle() returns file content when file exists
    - readGermanArticle() returns null when file does not exist
    - writeTranslatedArticle() writes file to docs/{lang}/{area}/{slug}.md
    - writeTranslatedArticle() creates parent directories recursively
    - writeTranslatedArticle() returns the relative path
