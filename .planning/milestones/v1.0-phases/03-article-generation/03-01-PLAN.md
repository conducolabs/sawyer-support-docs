    - Creates directory with `mkdirSync(dirname(absolutePath), { recursive: true })`

    - Returns the relative path string
    - Import Feature type from `../scanner/index.js`, buildArticlePath from `../paths/index.js`

    - Call `query()` with: prompt = buildFeaturePrompt(feature), options = { tools: [], permissionMode: 'bypassPermissions', allowDangerouslySkipPermissions: true, maxTurns: 3, model, systemPrompt: GENERATION_SYSTEM_PROMPT, settings: { modelSettings: { temperature: 0 } } }

    - After loop: prepend `buildFrontmatter(feature)` to articleBody
