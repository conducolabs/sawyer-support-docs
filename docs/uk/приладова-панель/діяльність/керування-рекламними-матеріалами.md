# Керування рекламними матеріалами

Ви можете створювати та керувати рекламними матеріалами для заходів, вакансій та контентом стартової сторінки.

## Відкрийте редактор рекламних матеріалів

Натисніть на опцію **Рекламний матеріал** для потрібної активності, вакансії або запису на стартовій сторінці.

## Редагування полів

Залежно від типу контенту доступні різні поля:

### Для заходів
- **Назва** (необов'язково) - Рекламна назва заходу.
- **Субтитри** (необов'язково) - додаткові субтитри.

### Для вакансій
- **Назва вакансії** (необов'язково) - рекламна назва вакансії.
- **Назва компанії** (необов'язково) - назва компанії.

### Для контенту головної сторінки
- **Заголовок** (необов'язково) - рекламний заголовок.
- **Subline** (необов'язково) - додатковий рядок.

## Мова

Ви можете використовувати перемикач мов для створення рекламних креативів різними мовами (німецькою та англійською).
