# Переглянути список очікування

Якщо захід повністю заброньовано, зацікавлені особи можуть бути внесені до списку очікування. Ви можете переглянути список очікування в будь-який час.

## Відкрити список очікування

Натисніть на **Лист очікування** для заходу, щоб переглянути поточний список.

## Відображена інформація

Для кожної людини в списку очікування ви можете побачити

- **Аватар** з ініціалами
- **Прізвище** людини
- **адресу електронної пошти** (на десктопних пристроях)
- **Дата реєстрації** - коли людина приєдналася до списку очікування

В кінці списку відображається **загальна кількість** осіб у черзі.
