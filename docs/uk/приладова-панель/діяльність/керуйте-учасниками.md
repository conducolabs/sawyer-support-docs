# Керуйте учасниками

Ви можете переглядати та керувати учасниками для кожної активності. Для цього натисніть на **Учасники** для потрібної активності.

## Список учасників

В огляді ви можете побачити всіх зареєстрованих учасників:

- Ім'я
- Статус (Підтверджено / Очікує)
- Статус оплати (Сплачено / Очікує / Безкоштовно)

Натисніть на учасника, щоб відкрити детальний перегляд.

## Додати учасника

1. Натисніть на **Додати учасника:в**.
2. **Крок 1: Введіть адресу електронної пошти учасника та натисніть на Пошук.
3. **Крок 2: Виберіть потрібних людей зі списку результатів. Також відображаються акаунти дітей.
   - Люди, які вже зареєстровані, підсвічуються.
   - Вікові індикатори показують, чи відповідає особа віковим обмеженням для участі у заході.
4. Натисніть на **Додати**, щоб завершити реєстрацію.

## Учасник: в деталях

У детальному поданні ви знайдете:

- **Персональні дані**: Ім'я, електронна пошта, телефон, дата народження та вік.
- **Дані про батьків**: Ім'я та номер телефону батьків (за наявності).
- **Алергія / Медична інформація**: Зареєстрована медична інформація.
- **Бейджик для плавання**: Вказівка на те, чи вміє людина плавати.
- **Інформація про оплату**: Статус оплати та спосіб оплати (наприклад, PayPal, кредитна картка, Apple Pay, оплата готівкою, MünsterlandCard, безкоштовно).

### Підтвердити платіж

Для готівкових платежів або платежів MünsterlandCard зі статусом "Створено" ви можете підтвердити платіж вручну. Для цього натисніть на **Підтвердити платіж** і підтвердіть дію в діалозі.

### Видалити учасника:в

Натисніть на **Видалити** і підтвердіть дію в діалозі підтвердження, щоб видалити учасника з активності. Це також призведе до відшкодування онлайн-платежів.

## Експортувати список учасників

Натисніть на кнопку **Завантажити**, щоб завантажити список учасників у форматі CSV. Перед експортом ви повинні підтвердити умови використання, поставивши галочку.
