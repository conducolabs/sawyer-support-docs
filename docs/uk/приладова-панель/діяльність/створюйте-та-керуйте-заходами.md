# Створюйте та керуйте заходами

У розділі **Заходи** в навігації клубу ви знайдете всі події вашого клубу. Сторінка розділена на дві вкладки: ***Активні заходи** та **Архівні заходи**.

## Створити захід

Натисніть на кнопку **Додати заняття**. У вас є два варіанти:

- **Режим ШІ** - Асистент conduco AI допоможе вам відповісти на запитання і створить подію автоматично (потребує кредитів ШІ).
- **Режим форми - ви заповнюєте форму події вручну.

### Поля форми

- **Зображення** (необов'язково) - завантажте зображення або створіть його за допомогою ШІ.
- **Назва** (обов'язково) - назва події.
- **Субтитри** (необов'язково) - необов'язковий субтитр.
- **Тізер** (обов'язково) - короткий опис події.
- **Опис** (обов'язкове поле) - Детальний опис у редакторі розширеного тексту.
- **Дати** (обов'язкове поле) - Принаймні одна дата із зазначенням дати початку та закінчення. Ви можете додати кілька дат, наприклад, для подій, що тривають кілька днів.
- **Місце проведення** (обов'язкове поле) - Виберіть існуюче місце проведення або створіть нове безпосередньо.
- **Місце зустрічі** (необов'язкове поле) - необов'язкове місце зустрічі для транспорту.
- **Категорія** (необов'язкове поле) - Приєднайте подію до категорії.
- **Бронювання з** (обов'язкове поле) - період, з якого учасники можуть реєструватися.
- **Місткість** (обов'язкове поле) - Максимальна кількість учасників (мінімум 1).
- **Ціна** (обов'язкове поле) - Вартість на одну особу в євро (мінімум 0).
- **Мінімальний вік** (обов'язкове поле) - мінімальний вік учасників (за замовчуванням: 8).
- **Максимальний вік** (обов'язкове поле) - максимальний вік учасників (за замовчуванням: 21).
- **Бронювання** - позначка, чи можна забронювати захід.
- **Онлайн-захід** - прапорець для онлайн-заходів. Якщо він активований, з'являється поле для посилання на конференцію.
- **Зовнішня реєстрація** - прапорець для реєстрації через зовнішню платформу. Якщо він активований, з'являється поле для зовнішнього посилання.
- **Лідер** (обов'язкове поле) - Принаймні один організатор повинен бути призначений. За замовчуванням ви вказуєте себе.

### Підтримка штучного інтелекту у формі

Окрім полів Назва, Тизер та Опис, ви знайдете кнопки ШІ, за допомогою яких ви можете:

- Генерувати тексти (якщо поле порожнє)
- Покращити існуючі тексти (якщо поле заповнене)

ШІ-генерація також доступна для зображення події.

## Редагування активності

Натисніть на **Редагувати** для активної події, щоб відкрити форму з існуючими даними. Зміни у зустрічах синхронізуються миттєво.

## Копіювати активність

Використовуйте функцію **Копіювати**, щоб створити новий захід на основі існуючого. Всі дані копіюються і можуть бути налаштовані.

## Архівні роботи

Ви можете знайти минулі заходи на вкладці **Архівні заходи**. Тут ви можете переглянути список учасників, але більше не можете редагувати подію.

## Пагінація

Події відображаються сторінка за сторінкою (25 на сторінці). Для переходу між сторінками використовуйте кнопки вперед/назад.
