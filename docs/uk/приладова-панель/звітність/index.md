# ♪ Звітність

У цьому розділі ви дізнаєтеся, як користуватися фінансовими звітами та статистикою вашого клубу. Тут ви дізнаєтеся, як аналізувати продажі в часі, оцінювати способи оплати, а також переглядати та експортувати детальну статистику активності.
