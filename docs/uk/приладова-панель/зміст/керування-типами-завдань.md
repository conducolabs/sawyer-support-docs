# Керування типами завдань

У розділі **Вміст > Типи вакансій** ви можете керувати доступними типами вакансій як супер-адміністратор. Вони пропонуються на вибір при створенні вакансій.

## Список типів вакансій

Список показує для кожного типу вакансії:

- **символ/зображення** типу роботи
- **Назва** типу завдання
- **Опис** (відображається лише на десктопних пристроях)
- **Видимість** - значок, що вказує на те, чи є тип завдання видимим або прихованим

На одній сторінці відображається 25 типів вакансій.

## Додати тип завдання

Натисніть на **Додати тип роботи**, щоб створити новий тип роботи.

## Редагувати тип завдання

Натисніть на кнопку **Редагувати**, щоб змінити існуючий тип вакансії. Ви можете налаштувати назву, опис, зображення та видимість.
