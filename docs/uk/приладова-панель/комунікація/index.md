# Спілкування

У цьому розділі ви дізнаєтеся, як спілкуватися зі своїми користувачами. Тут ви дізнаєтеся, як створювати, планувати та надсилати push-повідомлення, а також як створювати та керувати акціями зі скануванням QR-кодів.
