# Ролі та повноваження

Існують різні ролі на інформаційній панелі, які визначають, які області та функції доступні для вас.

## Супер адміністратор

Як супер-адміністратор, ви маєте доступ до всіх областей інформаційної панелі:

- Всі глобальні області адміністрування (користувачі, клуби, компанії)
- Всі області вмісту (стартова сторінка, категорії, типи вакансій)
- Об'єкти та MünsterlandCard
- Комунікація (сповіщення, акції)
- Налаштування програми, кредити ШІ та журнал активності
- Усі специфічні для клубу та компанії області

## Адміністратор клубу

Як адміністратор клубу ви можете:

- Переглядати інформаційну панель
- Керувати подіями для призначених клубів
- Переглядати звітність
- Керувати локаціями
- Редагувати профілі та налаштування клубів
- Керування профілями компаній та вакансіями

## Адміністратор компанії

Як адміністратор компанії ви можете:

- Переглядати інформаційну панель
- Керувати вакансіями вашої компанії
- Керувати контактними особами
- Редагувати профіль та налаштування компанії
