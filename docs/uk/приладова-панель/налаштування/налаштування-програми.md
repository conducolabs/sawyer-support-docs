# Налаштування програми

У розділі **Налаштування > Налаштування програми** ви можете налаштувати основні параметри програми як супер-адміністратор.

## Загальні налаштування

- **Imprint URL** (обов'язкове поле) - URL-адреса вашого відбитка. Повинна бути дійсною HTTPS-адресою.
- **Privacy URL** (обов'язкове поле) - URL-адреса вашої політики конфіденційності. Повинна бути дійсною HTTPS-адресою.
- **Загальні положення та умови** (обов'язкове поле) - Положення та умови вашого додатку. Редагується за допомогою текстового редактора з повним форматуванням.

Натисніть **Зберегти**, щоб застосувати зміни.

## Значення за замовчуванням для ШІ-кредитів

Тут ви встановлюєте кредитні квоти за замовчуванням:

- **Стандартні кредити для клубів** - скільки щомісячних кредитів ШІ отримує новий клуб за замовчуванням.
- **Стандартні кредити для компаній** - скільки щомісячних кредитів ШІ отримує нова компанія за замовчуванням.

Натисніть **Зберегти**, щоб оновити значення.

## Керування меню

Тут ви можете керувати пунктами меню, які відображаються в додатку.

### Додати пункт меню

Натисніть на **Додати пункт меню** і введіть наступну інформацію:

- **Назва** - текст пункту меню, що відображатиметься на екрані.
- **URL/Посилання** - цільова URL-адреса пункту меню.
- **Іконка** (необов'язково) - іконка для пункту меню.

### Порядок зміни

Пункти меню можна розташувати в потрібному порядку за допомогою **drag-and-drop**. Сортування зберігається автоматично.

### Редагувати пункт меню

Натисніть на **Редагувати**, щоб змінити дані пункту меню.

### Видалити пункт меню

Натисніть на **Видалити**, щоб видалити пункт меню.
