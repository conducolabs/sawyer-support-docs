# Налаштування

У цьому розділі ви знайдете всю інформацію про загальні налаштування. Дізнайтеся, як налаштувати юридичне повідомлення, захист даних, умови та положення, призначати та контролювати кредити ШІ, а також використовувати журнал активності для відстеження дій у системі.
