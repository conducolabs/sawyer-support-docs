# Адміністрація компанії

У цьому розділі ви дізнаєтеся, як керувати своєю компанією в панелі управління. Тут ви дізнаєтеся, як редагувати профіль компанії, налаштовувати параметри, створювати контактних осіб, а також створювати і підтримувати вакансії.
