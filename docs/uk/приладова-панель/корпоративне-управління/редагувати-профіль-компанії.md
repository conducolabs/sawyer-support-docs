# Редагувати профіль компанії

Ви можете редагувати профіль вашої компанії в розділі **Профіль** в навігації по компанії.

## Поля в профілі компанії

- **Назва компанії** (обов'язкове поле) - назва вашої компанії.
- **Опис** (обов'язкове поле) - опис вашої компанії.
- **Website** (обов'язкове поле) - Веб-адреса вашої компанії. Повинна починатися з `https://`.
- **Логотип** (необов'язкове поле) - Логотип компанії. Може бути завантажений через завантаження зображень.
- **Зображення обкладинки** (необов'язково) - зображення обкладинки компанії.

## Зберегти

Кнопка **Зберегти** активна тільки після того, як ви внесли зміни і правильно заповнили всі обов'язкові поля. Неправильно заповнені поля підсвічуються червоним кольором.
