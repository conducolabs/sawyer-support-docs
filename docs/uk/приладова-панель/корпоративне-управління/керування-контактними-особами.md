# Керування контактними особами

У розділі **Контактні особи** в навігації по компанії ви можете керувати контактними особами вашої компанії. Вони використовуються, наприклад, як контактні особи для вакансій.

## Додати контактну особу

Натисніть на **Додати контактну особу** і заповніть наступні поля:

- **Ім'я** (обов'язкове поле)
- **Прізвище** (обов'язкове поле)
- **Посада** (обов'язкове поле) - посада особи в компанії.
- **E-mail** (обов'язкове поле) - Дійсна адреса електронної пошти.
- **Телефон** (обов'язкове поле) - номер телефону контактної особи.
- **Фото профілю** (необов'язкове поле) - фотографія контактної особи.

## Редагувати контактну особу

Натисніть на **Редагувати** для потрібної особи, щоб змінити дані у формі.

## Видалити контактну особу

Ви також можете видалити контактну особу за допомогою функції редагування. Дія повинна бути підтверджена в діалоговому вікні підтвердження.

## Огляд

Всі контактні особи відображаються у списку з аватаром, ім'ям, посадою, електронною поштою та номером телефону.
