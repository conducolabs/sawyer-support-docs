# Reporting

In diesem Bereich erfährst du, wie du die Finanzberichte und Statistiken deines Vereins nutzt. Hier lernst du, wie du Umsätze im Zeitverlauf analysierst, Zahlungsmethoden auswertest und detaillierte Aktivitäts-Statistiken einsehen und exportieren kannst.
