# Vereinsverwaltung

In diesem Bereich erfährst du, wie du das Profil deines Vereins bearbeitest und die Vereinseinstellungen verwaltest. Hier lernst du, wie du Logo, Beschreibung und Kontaktdaten pflegst, Admins hinzufügst oder entfernst und die Sichtbarkeit sowie AGB deines Vereins konfigurierst.
