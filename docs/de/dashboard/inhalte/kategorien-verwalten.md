# Kategorien verwalten

Unter **Inhalte > Kategorien** verwaltest du als Super-Admin die Kategorien, in die Events eingeordnet werden können.

## Kategorie hinzufügen

Klicke auf **Kategorie hinzufügen** und gib den Namen der neuen Kategorie ein.

## Reihenfolge ändern

Die Kategorien können per **Drag-and-Drop** in die gewünschte Reihenfolge gebracht werden. Ziehe eine Kategorie an die gewünschte Position – die neue Sortierung wird automatisch gespeichert.

## Kategorie bearbeiten

Klicke auf den **Bearbeiten**-Button neben einer Kategorie, um den Namen zu ändern.

## Kategorie löschen

Klicke auf den **Löschen**-Button neben einer Kategorie und bestätige die Aktion.

## Leerer Zustand

Wenn noch keine Kategorien angelegt sind, wird ein Hinweis mit einer Aufforderung zum Erstellen der ersten Kategorie angezeigt.

## Sichtbarkeit von Kategorien

Kategorien können ein und ausgeblendet werden. Damit kann gesteuert werden, ob die Kategorie in der App sichtbar ist.

Zusätzlich dazu gibt es die Option **In Übersicht ausblenden**. Wird diese Option aktiviert, werden Aktivitäten ausschließlich in der Kategorie angezeigt in der App und nicht in der **Alle** Übersicht.

In der Kombination der beiden Filter können Veranstaltungen angelegt und zu einem bestimmten Punkt erst für alle Benutzer:innen der App angezeigt werden.
