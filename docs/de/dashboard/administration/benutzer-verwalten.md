# Benutzer:innen verwalten

Unter **Verwaltung > Benutzer:innen** findest du als Super-Admin die Übersicht aller registrierten Nutzer:innen.

## Benutzer:innen suchen

Nutze das Suchfeld oben auf der Seite, um nach Benutzer:innen zu suchen. Die Suche funktioniert nach Name und E-Mail-Adresse. Die Ergebnisse aktualisieren sich automatisch während der Eingabe.

## Benutzerliste

Die Liste zeigt pro Person:

- **Avatar** mit Initialen
- **Vorname** und **Nachname**
- **E-Mail-Adresse**

Auf mobilen Geräten wird die Liste als Karten-Ansicht dargestellt, auf Desktop-Geräten als Tabelle. Es werden 25 Einträge pro Seite angezeigt.

## Verfügbare Aktionen

Über das Aktionen-Menü (drei Punkte) bei jeder Person stehen dir folgende Funktionen zur Verfügung:

### Details anzeigen

Öffnet eine Detailansicht mit allen hinterlegten Informationen der Person.

### E-Mail ändern

Ändert die E-Mail-Adresse einer Person. Die neue Adresse muss gültig sein und darf maximal 255 Zeichen lang sein.

### Passwort-Reset senden

Sendet einen Link zum Zurücksetzen des Passworts an die hinterlegte E-Mail-Adresse. Die Aktion muss im Dialog bestätigt werden.

### Verifizierung erneut senden

Sendet die Verifizierungs-E-Mail erneut. Diese Option ist nur sichtbar, wenn die E-Mail-Adresse noch nicht verifiziert wurde.

### Rolle ändern

Ändert die Rolle einer Person. Wähle die neue Rolle aus dem Dropdown und bestätige die Änderung im Bestätigungsdialog.

### Aktivität anzeigen

Zeigt die Aktivitätshistorie der ausgewählten Person an.

### Kinderanzahl ändern

Ändert die maximale Anzahl an Kinder-Accounts, die eine Person anlegen kann. Der Wert muss mindestens 0 sein.

### MünsterlandCard ändern

Ermöglicht das Hinterlegen oder Ändern einer MünsterlandCard-Nummer. Diese Option ist nur verfügbar, wenn das MünsterlandCard-Feature aktiviert ist.
