# Warteliste einsehen

Wenn eine Aktivität ausgebucht ist, können sich Interessierte auf die Warteliste setzen lassen. Du kannst die Warteliste jederzeit einsehen.

## Warteliste öffnen

Klicke bei einer Aktivität auf **Warteliste**, um die aktuelle Liste einzusehen.

## Angezeigte Informationen

Für jede Person auf der Warteliste siehst du:

- **Avatar** mit Initialen
- **Name** der Person
- **E-Mail-Adresse** (auf Desktop-Geräten)
- **Anmeldedatum** – Wann sich die Person auf die Warteliste gesetzt hat

Am Ende der Liste wird die **Gesamtanzahl** der Personen auf der Warteliste angezeigt.
