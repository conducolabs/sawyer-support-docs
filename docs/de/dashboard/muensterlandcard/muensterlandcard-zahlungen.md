# MünsterlandCard-Zahlungen verwalten

Unter **Einrichtungen & Orte > MünsterlandCard** verwaltest du als Super-Admin alle MünsterlandCard-Zahlungen. Diese Funktion ist nur verfügbar, wenn das MünsterlandCard-Feature für deinen Account aktiviert ist.

## Zeitraum filtern

Wähle über die Datumsfelder den gewünschten Zeitraum:

- **Startdatum** – Ab welchem Datum Zahlungen angezeigt werden.
- **Enddatum** – Bis zu welchem Datum Zahlungen angezeigt werden.

## Zahlungsliste

Die Liste zeigt pro Eintrag:

- **Benutzername** – Name der zahlenden Person.
- **Kartennummer** – Die hinterlegte MünsterlandCard-Nummer. Falls keine Nummer hinterlegt ist, wird ein Warnhinweis angezeigt.
- **Event-Name** – Für welches Event bezahlt wurde.
- **Event-Datum** – Datum und Uhrzeit des Events.
- **Preis** – Der Zahlungsbetrag in Euro.
- **Zahlungsstatus** – Badge mit dem Status (BEZAHLT oder ERSTELLT).
- **Genehmigungsstatus** – Badge mit dem Status (Genehmigt oder Ausstehend).

Es werden 25 Einträge pro Seite angezeigt. Auf Mobilgeräten wird die Liste als Karten dargestellt.

## Aktionen

Über das Aktionen-Menü bei jedem Eintrag stehen dir folgende Optionen zur Verfügung:

### Genehmigen

Genehmigt eine ausstehende MünsterlandCard-Zahlung. Diese Option ist nur verfügbar, wenn eine Kartennummer hinterlegt ist und die Zahlung noch nicht genehmigt wurde. Die Aktion muss im Bestätigungsdialog bestätigt werden.

### Ablehnen

Lehnt eine ausstehende Zahlung ab. Die Aktion muss im Bestätigungsdialog bestätigt werden.

### Kartennummer hinzufügen / ändern

Öffnet einen Dialog, in dem du die MünsterlandCard-Nummer eingeben oder aktualisieren kannst.

## CSV-Export

Klicke auf den **Export**-Button, um die aktuelle Ansicht als CSV-Datei herunterzuladen.
