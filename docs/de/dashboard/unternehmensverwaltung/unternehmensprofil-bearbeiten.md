# Unternehmensprofil bearbeiten

Unter **Profil** in der Unternehmens-Navigation bearbeitest du das Profil deines Unternehmens.

## Felder im Unternehmensprofil

- **Unternehmensname** (Pflichtfeld) – Der Name deines Unternehmens.
- **Beschreibung** (Pflichtfeld) – Eine Beschreibung deines Unternehmens.
- **Website** (Pflichtfeld) – Die Webadresse deines Unternehmens. Muss mit `https://` beginnen.
- **Logo** (optional) – Das Unternehmenslogo. Kann über den Bild-Upload hochgeladen werden.
- **Titelbild** (optional) – Ein Titelbild für das Unternehmen.

## Speichern

Der **Speichern**-Button ist erst aktiv, wenn du Änderungen vorgenommen hast und alle Pflichtfelder korrekt ausgefüllt sind. Fehlerhafte Felder werden rot hervorgehoben.
