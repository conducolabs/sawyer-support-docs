# Şirket yönetimi

Bu bölümde, kontrol panelinde şirketinizi nasıl yöneteceğinizi öğreneceksiniz. Burada şirket profilini nasıl düzenleyeceğinizi, ayarları nasıl yapılandıracağınızı, irtibat kişilerini nasıl oluşturacağınızı ve iş ilanlarını nasıl oluşturup koruyacağınızı öğreneceksiniz.
