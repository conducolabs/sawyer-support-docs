# Şirketi yönetin

Bu bölümde merkezi yönetimle ilgili tüm bilgileri bulacaksınız. Süper yönetici olarak burada kullanıcıları yönetebilir, kulüpler oluşturup etkinleştirebilir ve şirketler oluşturup yapılandırabilirsiniz.
