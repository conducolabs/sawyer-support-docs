# MünsterlandCard

Bu bölümde MünsterlandCard ödemelerini nasıl yöneteceğinizi öğreneceksiniz. Burada ödemeleri nasıl görüntüleyeceğinizi, yetkilendireceğinizi veya reddedeceğinizi, kart numaralarını nasıl koruyacağınızı ve ödeme verilerini CSV olarak nasıl dışa aktaracağınızı öğreneceksiniz.
