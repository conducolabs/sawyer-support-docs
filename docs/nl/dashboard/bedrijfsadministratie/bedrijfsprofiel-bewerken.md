# Bedrijfsprofiel bewerken

Je kunt het profiel van je bedrijf bewerken onder **Profiel** in de bedrijfsnavigatie.

## Velden in het bedrijfsprofiel

- **Bedrijfsnaam** (verplicht veld) - De naam van je bedrijf.
- **Beschrijving** (verplicht veld) - Een beschrijving van je bedrijf.
- **Website** (verplicht) - Het webadres van je bedrijf. Moet beginnen met `https://`.
- **Logo** (optioneel) - Het bedrijfslogo. Kan worden geüpload via de afbeelding-upload.
- **Coverafbeelding** (optioneel) - Een coverafbeelding voor het bedrijf.

## Opslaan

De knop **Opslaan** is pas actief nadat je wijzigingen hebt aangebracht en alle verplichte velden correct zijn ingevuld. Onjuiste velden worden rood gemarkeerd.
