# Clubbestuur

In dit onderdeel leer je hoe je het profiel van je club kunt bewerken en de clubinstellingen kunt beheren. Hier leer je hoe je het logo, de beschrijving en de contactgegevens kunt onderhouden, hoe je admins kunt toevoegen of verwijderen en hoe je de zichtbaarheid en de algemene voorwaarden van je club kunt configureren.
