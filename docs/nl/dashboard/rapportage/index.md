# Reporting

In dit onderdeel leer je hoe je de financiële rapporten en statistieken van je club kunt gebruiken. Hier leer je hoe je de verkopen in de loop van de tijd kunt analyseren, betaalmethoden kunt evalueren en gedetailleerde activiteitsstatistieken kunt bekijken en exporteren.
