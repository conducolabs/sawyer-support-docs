# Dashboard

In dit gedeelte vind je alle informatie over het admin dashboard. Ontdek hoe je je weg kunt vinden in het dashboard, clubs en bedrijven kunt beheren, activiteiten kunt aanmaken, inhoud kunt onderhouden en de app-instellingen kunt configureren. De artikelen zijn per onderwerp ingedeeld zodat je snel de hulp kunt vinden die je nodig hebt.
