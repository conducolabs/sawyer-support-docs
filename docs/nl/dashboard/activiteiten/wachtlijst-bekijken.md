# Wachtlijst bekijken

Als een activiteit volgeboekt is, kunnen geïnteresseerden op de wachtlijst worden geplaatst. Je kunt de wachtlijst op elk moment bekijken.

## Wachtlijst openen

Klik op **Wachtlijst** voor een activiteit om de huidige lijst te bekijken.

## Weergegeven informatie

Voor elke persoon op de wachtlijst kun je het volgende zien

- **Avatar** met initialen
- **Naam** van de persoon
- **E-mailadres** (op desktopapparaten)
- **Inschrijfdatum** - wanneer de persoon op de wachtlijst is gekomen

Het **totaal aantal** personen op de wachtlijst wordt aan het einde van de lijst weergegeven.
