# Activiteiten

In dit gedeelte vind je alles wat je moet weten over het zoeken naar en registreren voor activiteiten. Je leert hier hoe je evenementen kunt vinden en filteren, wat de detailpagina laat zien, hoe het registratieproces werkt en welke betalingsmethoden beschikbaar zijn. Je kunt ook te weten komen hoe je jezelf op een wachtlijst kunt zetten als een activiteit volgeboekt is.
