Hier lees je hoe je kinderprofielen kunt aanmaken, bewerken en verwijderen uit je account. Met kindprofielen kun je je kinderen direct via je account inschrijven voor activiteiten. Een kindprofiel kan ook worden omgezet in een onafhankelijk account als dat nodig is.
