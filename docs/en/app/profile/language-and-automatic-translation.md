## Overview

The app supports multiple languages. You can change the language of the user interface and activate automatic translation for content.

## Step-by-step

### Change app language

1. Open **Edit profile** via your user menu.
2. Scroll to the **Language** section.
3. Tap on **App language** and select your preferred language:
   - German
   - English
   - Dutch
   - Türkçe
   - Українська
4. The language is updated immediately.

### Automatic translation

If you select a language other than German, content such as activity descriptions and news can be automatically translated into your selected language. Automatically translated content is marked with the note **Translated with conduco AI (Beta)**.

The translation only affects the content (e.g. descriptions and news), not the app's user interface.

## FAQ

**How good is the automatic translation?
The translation is generated by an AI and is still in the beta phase. In most cases, it delivers good results, but inaccuracies may occur occasionally.

**Can I deactivate the automatic translation?
Yes, switch the app language back to the original language of the content (usually German) to deactivate the translation.
