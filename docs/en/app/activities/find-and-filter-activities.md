## Overview

In the **Activities** tab you will find all upcoming events and offers of your youth center. You can filter the list by category to quickly find what you are looking for.

## Step-by-step

1. Tap on the **Activities** tab in the bottom navigation.
2. You will see a list of all available activities.
3. You can narrow down the selection using the category filters at the top. Tap on **All** to display all activities again.
4. Scroll through the list. More activities are loaded automatically.
5. Drag the list down to load the latest activities.
6. Tap on an activity to display the details.

Each card shows you the most important information at a glance: Image, title, date and price. If an activity is already full, it will be marked as **booked out**.

## FAQ

**Why don't I see any activities?
If "No activities" is displayed, there are currently no upcoming offers. Drag the list down to refresh.

**Can I search for activities?**
Currently you can narrow down the activities using the category filters. A text search is not available in the activities tab.
