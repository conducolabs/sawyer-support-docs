## Overview

You can update your children's data or remove a child profile from your account at any time.

## Step-by-step

### Edit child profile

1. Open **Manage children** via your user menu.
2. Tap on the child profile you want to edit.
3. Change the desired fields (name, address, medical information, swimming badge).
4. Tap on **Save**.

### Remove child profile

1. Open **Manage children** via your user menu.
2. Tap on the child profile you want to remove.
3. Tap on **Delete**.
4. Confirm the removal. The following message will appear: "This will remove [name] from your account."

## FAQ

**What happens to existing registrations when I remove a child?
Existing registrations for activities remain unaffected. The child will simply be disconnected from your account.

**Can I add a removed child again?
Yes, you can add a child as a new profile at any time.
