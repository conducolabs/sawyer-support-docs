# Manage notifications

Under **Communication > Notifications** you can manage push notifications to users as a super admin. This function is only available if it is activated for your account.

## Notification list

The overview shows all notifications with:

- **Title** of the notification
- **Status** - Color-coded badge:
  - Draft
  - Planned
  - Sent
  - Sent
  - Failed
- **Number of recipients**
- **Date** (sent, scheduled or created)
- **Created by** - Name of the person who created it

## Create notification

Click on **Add notification**. You will be redirected to the form page.

### Content

- **Title** (mandatory field) - The title of the notification.
- **Text** (mandatory field) - The content of the message. A character count indicates the current text length.
- **Image** (optional) - An optional image for the notification.
- **Internal link** (optional) - Select a target page in the app:
  - News
  - Events
  - Jobs
  - Clubs
  - Facilities
  - Customized link

### Target group (segment)

Define who should receive the notification:

- **Minimum age** (optional) - recipients over a certain age.
- **Maximum age** (optional) - recipients up to a certain age.
- **Min. activities** (optional) - minimum number of events attended.
- **Max. Activities** (optional) - Maximum number of events attended.
- **Registered since** (optional) - Only people who have been registered since a certain date.
- **Device type** - All devices, iOS only or Android only.
- **Role** (optional) - Only people with a specific role.

The estimated **recipient count** is displayed dynamically and updates when you change the filters.

## Send and schedule

You have the following options:

- **Send test** - Sends a test notification to your own device.
- **Save as draft** - Saves the notification as a draft.
- **Schedule** - Schedule the notification to be sent at a specific time using the date/time picker.
- **Send now** - Sends the notification immediately. The action must be confirmed in the confirmation dialog.

## Edit notification

Drafts and scheduled notifications can be edited. To do this, click on **Edit** in the Actions menu.

## Delete notification

Drafts and scheduled notifications can be deleted. Notifications that have already been sent can only be viewed.
