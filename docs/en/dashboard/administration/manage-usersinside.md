# Manage users:inside

Under **Administration > Users** you will find an overview of all registered users as a super admin.

## Search for users

Use the search field at the top of the page to search for users. The search works by name and email address. The results update automatically as you type.

## User list

The list shows per person:

- **Avatar** with initials
- **first name** and **last name**
- **e-mail address**

On mobile devices the list is displayed as a card view, on desktop devices as a table. 25 entries are displayed per page.

## Available actions

The following functions are available via the Actions menu (three dots) for each person:

### Show details

Opens a detailed view with all the person's stored information.

### Change e-mail

Changes a person's e-mail address. The new address must be valid and may be a maximum of 255 characters long.

### Send password reset

Sends a link to reset the password to the stored e-mail address. The action must be confirmed in the dialog.

### Send verification again

Sends the verification email again. This option is only visible if the e-mail address has not yet been verified.

### Change role

Changes the role of a person. Select the new role from the dropdown and confirm the change in the confirmation dialog.

### Show activity

Shows the activity history of the selected person.

### Change number of children

Changes the maximum number of child accounts that a person can create. The value must be at least 0.

### Change MünsterlandCard

Allows you to store or change a MünsterlandCard number. This option is only available if the MünsterlandCard feature is activated.
