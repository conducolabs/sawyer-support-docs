# Manage categories

Under **Content > Categories** you as super admin manage the categories in which events can be categorized.

## Add category

Click on **Add category** and enter the name of the new category.

## Change order

The categories can be put in the desired order using **drag-and-drop**. Drag a category to the desired position - the new sorting will be saved automatically.

## Edit category

Click on the **Edit** button next to a category to change the name.

## Delete category

Click on the **Delete** button next to a category and confirm the action.

## Empty state

If no categories have been created yet, a message is displayed asking you to create the first category.

## Visibility of categories

Categories can be shown and hidden. This allows you to control whether the category is visible in the app.

In addition, there is the option **Hide in overview**. If this option is activated, activities are only displayed in the category in the app and not in the **All** overview.

In the combination of the two filters, events can be created and only displayed for all users of the app at a certain point.
