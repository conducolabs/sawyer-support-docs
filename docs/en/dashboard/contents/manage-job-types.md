# Manage job types

Under **Content > Job types** you can manage the available job types as a super admin. These are offered as a selection when creating job offers.

## Job type list

The list shows per job type:

- **symbol/image** of the job type
- **Title** of the job type
- **Description** (only visible on desktop devices)
- **Visibility** - badge indicating whether the job type is visible or hidden

25 job types are displayed per page.

## Add job type

Click on **Add job type** to create a new job type.

## Edit job type

Click on the **Edit** button to change an existing job type. You can change the title, description, image and visibility.
