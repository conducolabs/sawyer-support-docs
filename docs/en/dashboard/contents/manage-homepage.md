# Manage start page

Under **Content > Start page**, you as super admin manage the content that is displayed on the start page of the app. The page is divided into two tabs: **Active** and **Archived**.

## Content types

There are four types of home page content:

- **Contribution (post)** - Editorial content
- **Activity** - Event information
- **Jobs** - Job offers
- **Promotion** - A promotion that you have created

Each content type is marked with a color-coded badge.

## Add content

Click on **Add content** to create a new entry. The following fields are available:

- **Heading** - The main heading of the entry.
- **Subline** - A supplementary line under the heading.
- **Type** - Choose between post, event or jobs.
- **Image** - An optional image for the entry.
- **Display from** - From which date the entry will be displayed.
- **Display until** - Until which date the entry is visible.

## Change order

The entries can be put in the desired order using **drag-and-drop**. Drag an entry to a new position and the sorting will be saved automatically.

## Edit content

Click on **Edit** to change an existing entry.

## Advertising material

You can use the **Advertising media** option to create and manage advertising media for the start page entry.

## Delete content

Click on **Delete** and confirm the action in the dialog.

## Archived content

You will find expired content in the **Archived** tab. This can be edited, provided with advertising material or deleted.
