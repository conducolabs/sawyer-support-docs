# Use AI functions

The dashboard offers AI-supported functions from conduco AI in various places to help you create content. AI credits are required for use.

## Where can you find AI functions?

### Activity creation

- **AI activity creation** - When creating a new activity, you can select the AI mode. An assistant will guide you through specific questions and create the event automatically. You can check the generated content, adjust it and regenerate it if necessary.
- **Field improvement** - You will find the conduco AI button in the Name, Teaser and Description fields:
  - **When field is empty**: The wizard asks you questions and generates the text.
  - **When the field is filled in**: The assistant analyzes the existing text and suggests improvements.
- **Image generation** - You can have an activity image generated by AI based on the title, teaser and category.

### Club profile

- **Description** - Generate or improve the club description via AI.
- **Cover picture** - Generate a cover picture based on the club name and description.

## How the AI text generation works

1. Click on the **conduco AI button** next to a text field.
2. The wizard will ask you specific questions about the desired content.
3. You can skip optional questions with **Skip**.
4. The wizard generates the text.
5. You can accept the result:
   - **Accept** - The text is inserted into the field.
   - **Generate new** - Give feedback and the wizard will create a new version.

## How the AI image generation works

1. Click on the **conduco AI button** next to the image field.
2. The system generates an image based on the text data entered (e.g. title, teaser).
3. Check the preview and click on **Apply** to use the image.
4. If necessary, you can use Feedback to generate a new version.

## AI Credits

Each AI generation consumes credits from your quota. You can see the current consumption:

- In the AI chat header as a badge.

Credits are allocated by super admins and can be allocated on a monthly or one-off basis.
