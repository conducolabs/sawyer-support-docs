# Activity log

Under **Settings > Activity log** you can track all actions in the system as a super admin.

## Filter

### Period

- **From** - Select a start date via the calendar. The last 7 days are set by default.
- **To** - Select an end date. Today's date is set by default.

### Categories

You can use the category filter to filter by specific activity types:

- Activate or deactivate individual categories via checkboxes.
- The display shows how many categories are currently selected (e.g. "3 / 8").
- You can select several categories at the same time.

## Activity list

Is displayed for each activity:

- **Type/Category** - With colored icon.
- **Description** - What exactly happened.
- **User:in** - Who performed the action.
- **Timestamp** - When the action took place.
- **Organization** - Which association or company is affected (if applicable).

## Update

Click on the **Refresh** button to load the latest activities. A loading spinner will be displayed during the loading process.

## Pagination

The activities are displayed page by page (15 per page). Use the forward/back buttons to navigate between the pages.
