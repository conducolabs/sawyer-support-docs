# View waiting list

If an activity is fully booked, interested parties can be placed on the waiting list. You can view the waiting list at any time.

## Open waiting list

Click on **Waiting list** for an activity to view the current list.

## Displayed information

For each person on the waiting list you can see

- **Avatar** with initials
- **Name** of the person
- **Email address** (on desktop devices)
- **Registration date** - when the person joined the waiting list

At the end of the list, the **total number** of people on the waiting list is displayed.
