# Create and manage activities

Under **Activities** in the club navigation you will find all your club's events. The page is divided into two tabs: **Active activities** and **Archived activities**.

## Create activity

Click on the **Add activity** button. You have two options:

- **AI mode** - The conduco AI assistant guides you through questions and creates the event automatically (requires AI credits).
- **Form mode** - You fill out the event form manually.

### Form fields

- **Image** (optional) - Upload an image or generate one via AI.
- **Name** (mandatory) - The title of the event.
- **Subtitle** (optional) - An optional subtitle.
- **Teaser** (mandatory) - A short description of the activity.
- **Description** (mandatory field) - The detailed description in the rich text editor.
- **Dates** (mandatory field) - At least one date with start and end date. You can add several dates, e.g. for events lasting several days.
- **Location** (mandatory field) - Select an existing location or create a new one directly.
- **Pick-up location** (optional) - An optional meeting point for transportation.
- **Category** (optional) - Assign the event to a category.
- **Bookable from** (mandatory field) - From when participants can register.
- **Capacity** (mandatory field) - Maximum number of participants (minimum 1).
- **Price** (mandatory field) - Cost per person in euros (minimum 0).
- **Minimum age** (mandatory field) - Minimum age for participants (default: 8).
- **Maximum age** (mandatory field) - Maximum age for participants (default: 21).
- **Bookable** - Checkbox whether the event is bookable.
- **Online event** - checkbox for online events. If activated, a field for the conference link appears.
- **External registration** - Checkbox for registration via an external platform. If activated, a field for the external link appears.
- **Leader** (mandatory field) - At least one organizer must be assigned. By default, you are entered yourself.

### AI support in the form

In addition to the Name, Teaser and Description fields, you will find AI buttons with which you can:

- Generate texts (if the field is empty)
- Improve existing texts (if the field is filled in)

AI generation is also available for the event image.

## Edit activity

Click on **Edit** for an active event to open the form with the existing data. Changes to the appointments are synchronized immediately.

## Copy activity

Use the **Copy** function to create a new activity based on an existing one. All data is copied and can be adjusted.

## Archived activities

You can find past events in the **Archived activities** tab. Here you can view the list of participants, but you can no longer edit the event.

## Pagination

Events are displayed page by page (25 per page). Use the forward/back buttons to navigate between the pages.
