# Dashboard overview

After logging in, you will land directly on the dashboard home page. Depending on your role, you will see different information here.

## Welcome

The dashboard greets you with "Good morning", "Good afternoon" or "Good evening", depending on the time of day.

## Statistics cards (only for super admins)

As a super admin, you will see several statistics cards with key figures on the start page:

- **Active activities** - number of currently active events, incl. comparison to the previous month.
- **Registrations for activities** - Registrations in the current month compared to the previous month.
- **Users** - Total number of registered users, including new additions in the last month.
- **Clubs** - Number of all clubs, including new ones.
- **Companies** - Number of all companies, including new additions.
- **Job vacancies** - Number of all jobs, including newly created jobs.

## Activity stream

An activity stream is displayed below the statistics, which summarizes the last actions in the system.

## View for club and company users

If you are not a super admin, you will be shown an individual content area instead of the statistics.
