# Edit company profile

You can edit your company's profile under **Profile** in the company navigation.

## Fields in the company profile

- **Company name** (mandatory field) - The name of your company.
- **Description** (mandatory field) - A description of your company.
- **Website** (required) - The web address of your company. Must start with `https://`.
- **Logo** (optional) - The company logo. Can be uploaded via the image upload.
- **Cover image** (optional) - A cover image for the company.

## Save

The **Save** button is only active once you have made changes and all mandatory fields have been filled in correctly. Incorrect fields are highlighted in red.
